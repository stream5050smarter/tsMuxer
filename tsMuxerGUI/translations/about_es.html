tsMuxeR es un programa para crear archivos TS y M2TS emitibles a través de 'streaming', también como para visionarlos mediante reproductores de vídeo (p. ej., Dune HD Ultra, Sony Playstation 3, Samsung Smart TV y otros).

tsMuxeR es software libre.

Soporta los siguientes formatos de entrada:

  • TS
  • M2TS
  • SIFF
  • MOV
  • MP4
  • MKV
  • Blu-ray
  • Opción de desmultiplexado

Códecs de vídeo soportados:

  • H.264/MVC
  • H.265/HEVC
  • Microsoft VC-1
  • MPEG-2

Códecs de audio soportados:

  • AAC
  • AC3/E-AC3(DD+)
  • Dolby True HD (solo para flujos con AC3)
  • DTS/DTS-HD
  • LPCM

Tipos de subtítulos soportados:

  • M2TS
  • SRT (subtítulos de texto)

Formatos y contenedores soportados:

  • MPEG (Elementary stream)
  • TS/M2TS (Transport stream)
  • EVO/VOB/MPG (Program stream)
  • MKV/MKA (Matroska)
  • MP4/MOV

Características principales:

  • Soporte para Blu-ray 3D
  • Soporte para UHD, HDR10, HDR10+ y Dolby Vision
  • Ajuste automático o manual de fps al entremezclar
  • Cambio de nivel, también como elementos SEI, SPS/PPS y unidad NAL delimitadora de inserción de ciclos al entremezclar vídeo en H.264
  • Modificación de tiempos para audios y subtítulos
  • Capacidad para extraer DTS a partir de DTS-HD
  • Capacidad para extraer AC3 a partir de True-HD
  • Capacidad para unir archivos
  • Capacidad para ajustar fps para subtítulos
  • Capacidad para convertir flujos LPCM en WAVE y viceversa
  • Capacidad para introducir información sobre el idioma de la pista en la estructura del Blu-ray y en la cabecera TS
  • Capacidad para cortar archivos de origen
  • Capacidad para dividiar archivos de salida
  • Capacidad para detectar retraso de audio en fuentes TS/M2TS/MPG/VOB/EVO
  • Capacidad para eliminar información de 'pulldown' del flujo de datos
  • Capacidad para abrir listas de reproducción Blu-ray (archivos .mpls)
  • Capacidad para convertir subtítulos SRT en PGS
  • Soporte para etiquetas de subtítulos SRT (etiquetas para cambiar la fuente, el color, el tamaño, etc.; la sintaxis de las etiquetas es similar a la de HTML)
  • Interfaz multiplataforma unificada para Windows, Linux y MacOS.
