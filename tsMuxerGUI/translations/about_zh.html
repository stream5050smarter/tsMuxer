tsMuxeR – 该程序可创建用于IP广播以及在硬件视频播放器（即Dune HD Ultra，Sony Playstation3，Samsung Smart TV等）上观看的TS和M2TS文件。

tsMuxeR是开源的。

支持的源文件格式：

  • TS;
  • M2TS;
  • SIFF;
  • MOV;
  • MP4;
  • MKV;
  • 蓝光;
  • Demux选项。

支持的视频编解码器：

  • H.264/MVC
  • H.265/HEVC;
  • Microsoft VC-1;
  • MPEG-2.

支持的音频编解码器：

  • AAC;
  • AC3 / E-AC3(DD+);
  • Dolby True HD (仅适用于具有AC3核心的流);
  • DTS/ DTS-HD;
  • LPCM.

支持的字幕类型：

  • M2TS图形字幕流(PGS)。
  • SRT文本字幕

支持的容器和格式：

  • 基本流;
  • 传输流TS和M2TS;
  • 节目流EVO/VOB/MPG;
  • Matroska MKV/MKA.
  • MP4/MOV.

主要特点：

  • 3D蓝光支持;
  • UHD, HDR10, HDR10+ 和 Dolby Vision 支持;
  • 混合时自动或手动fps调整;
  • 混合H.264时进行电平更改以及SEI，SPS/PPS元素和NAL单位定界符循环插入;
  • 音频和字幕轨道相对于视频轨道的时间偏移;
  • 从DTS-HD提取DTS核心;
  • 从True-HD提取AC3内核;
  • 加入文件;
  • 调整字幕的fps;
  • 能够将LPCM流转换为WAVE，反之亦然;
  • 轨道语言信息注入到蓝光结构和TS标头中;
  • 修剪源文件;
  • 分割输出文件;
  • 检测TS/M2TS/MPG/VOB/EVO源的音频延迟;
  • 从流中删除Pulldown信息;
  • 打开蓝光播放列表(MPLS)文件;
  • 将文本SRT字幕转换为图形字幕 PGS (SUP);
  • 支持SRT字幕的标签-用于更改字体，颜色，大小等标签； 标签的语法类似于HTML;
  • 统一的跨平台GUI - Windows, Linux, MacOS。
